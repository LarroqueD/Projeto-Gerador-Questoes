# Gerador de Estudos IA

## Sobre o projeto

O Gerador de Estudos IA é uma aplicação web que utiliza um modelo de linguagem (LLM) via API para criar materiais de estudo personalizados com base nos temas informados pelo usuário.

O objetivo do projeto é auxiliar estudantes na aprendizagem e revisão de conteúdos, gerando resumos, questões e explicações de forma automática. A ferramenta busca transformar um simples tema de estudo em um material completo para apoiar a preparação para provas, trabalhos e atividades acadêmicas.

## Como funciona

O usuário informa dados como:

* Tema ou assunto de estudo
* Nível de dificuldade desejado
* Quantidade de questões
* Tipo de atividade (objetiva, discursiva ou mista)

A aplicação processa essas informações e envia uma solicitação para a IA, que atua como um professor virtual especializado na criação de conteúdos educacionais.

Com base nisso, o sistema retorna:

* Resumo do conteúdo
* Questões de revisão
* Gabarito comentado
* Explicações didáticas
* Sugestões de tópicos relacionados para aprofundamento

## Objetivo acadêmico

Este projeto foi desenvolvido para demonstrar a integração entre uma aplicação backend e uma API de inteligência artificial, mostrando como LLMs podem ser utilizados para criar ferramentas educacionais inteligentes.

Além disso, o projeto evidencia como a engenharia de prompts pode direcionar o comportamento da IA para gerar conteúdos personalizados e úteis, transformando modelos de linguagem em funcionalidades práticas para sistemas reais.
